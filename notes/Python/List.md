# List

## Insert at end

``my_list.append(10)``

## Insert at Begining

``my_list.add(0, 10)``

## Inser at any index

``my_list.add(3, 10)``

## Sub Array

``my_list[i:j]``

## Reverse

``my_list[::-1]``

## Skip elements
---
``my_list[1::2]``
This will get u sub array from ``1`` to ``len(my_list)`` and it will ``skip`` each ``2nd`` element.
---
eg. 
my_list = [1,2,3,4,5,6]
my_list[1:5:2]
>>> [2,4]
